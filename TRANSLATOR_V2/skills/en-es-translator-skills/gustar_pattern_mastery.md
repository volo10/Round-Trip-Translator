# Gustar Pattern Mastery Skill

## Objective

Master the conversion of English "I like/love + [verb/noun]" structures to natural Spanish gustar constructions, avoiding the common error of direct translation.

## Critical Problem

**English:** "I like + gerund/noun"
**Spanish mistake:** "Yo gusta..." ← WRONG (violates Spanish grammar)
**Spanish correct:** "Me gusta..." ← RIGHT (uses indirect object)

### Why This Matters

The English structure "I like X" doesn't map directly to Spanish. Spanish uses an indirect object construction where the activity/object becomes the grammatical subject and the person becomes the indirect object.

## Core Pattern

### Structure: Indirect Object + Gusta Verb + Subject Activity

```
Me        gusta        ir a la playa
(to me)   (it pleases) (to go to beach)
= "I like going to the beach"
```

### Indirect Object Pronouns

| English | Spanish Indirect | Example | Full Translation |
|---------|------------------|---------|-----------------|
| I like | me gusta | Me gusta bailar | I like dancing |
| You like (informal) | te gusta | Te gusta correr | You like running |
| He/She likes | le gusta | Le gusta leer | He/She likes reading |
| We like | nos gusta | Nos gusta cantar | We like singing |
| You like (formal) | le gusta | Le gusta escribir | You (formal) like writing |
| They like | les gusta | Les gusta jugar | They like playing |

## Conjugation Rules

### Verb Conjugation for Gustar

The verb "gustar" always conjugates based on the **activity**, NOT the person liking it.

- **Singular activity:** uses 3rd person singular = **gusta**
  - "Me gusta ir" (I like to go) - one activity
  - "Te gusta bailar" (You like to dance) - one activity

- **Plural activities:** uses 3rd person plural = **gustan**
  - "Me gustan los perros" (I like dogs) - multiple items
  - "Te gustan las flores" (You like flowers) - multiple flowers

### Infinitive Form (No Conjugation)

When gustar is followed by an infinitive, the infinitive doesn't change:

```
Me gusta ir        (I like to go)
Me gusta correr    (I like to run)
Me gusta nadar     (I like to swim)
No me gusta leer   (I don't like to read)
```

## Application Examples

### Example 1: Simple Activity
```
English: "I like eating pizza"
Structure: Me + gusta + comer pizza
Spanish: "Me gusta comer pizza"
```

### Example 2: Multiple Items
```
English: "I like cats"
Structure: Me + gustan (plural) + los gatos
Spanish: "Me gustan los gatos"
Note: Use "gustan" because gatos is plural
```

### Example 3: Negation
```
English: "I don't like swimming"
Structure: No + me + gusta + nadar
Spanish: "No me gusta nadar"
Note: Negation comes at the beginning
```

### Example 4: Question
```
English: "Do you like dancing?"
Structure: ¿Te + gusta + bailar?
Spanish: "¿Te gusta bailar?"
```

## Common Errors to Avoid

### Error 1: Using Present Tense of Querer

❌ "Yo quiero comer pizza" (literal: "I want to eat pizza")
✓ "Me gusta comer pizza" (natural: "I like eating pizza")

**Why:** While both can be correct, gustar is more natural for simple preference.

### Error 2: Wrong Pronoun Conjugation

❌ "Yo me gusta" (mixing direct subject + indirect object)
✓ "Me gusta" (indirect object only)

**Why:** Gustar always uses the indirect object construction. Never use the subject pronoun "yo" with gustar.

### Error 3: Forgetting Infinitive Form

❌ "Me gusta corriendo" (using gerund instead of infinitive)
✓ "Me gusta correr" (using infinitive)

**Why:** Always use infinitive (-ar, -er, -ir) form after gustar, not gerund.

### Error 4: Wrong Number Agreement

❌ "Me gusta los gatos" (mixing singular gusta with plural nouns)
✓ "Me gustan los gatos" (plural gustan with plural nouns)

**Why:** The verb must agree with the object's number.

## Usage Context Rules

### When to Use Gustar

| Context | Spanish | English |
|---------|---------|---------|
| Preference | Me gusta el café | I like coffee |
| Simple activity | Me gusta bailar | I like dancing |
| Frequency | Me gusta a menudo | I like it often |
| Negation | No me gusta | I don't like |
| Strong preference | Me encanta | I love (emphatic) |

### Gusta vs. Quiero

| Structure | Example | Meaning |
|-----------|---------|---------|
| Me gusta... | Me gusta ir | I like to go / I like it |
| Quiero... | Quiero ir | I want to go (more direct desire) |

**Usage:** Gustar = passive preference, Quiero = active desire

## Pronoun Placement Rules

### With Gustar Verb

The indirect object pronoun **always goes BEFORE** the conjugated gustar verb:

```
Te gusta nadar
↑
Pronoun comes first
```

## Special Cases

### With Reflexive-Like Verbs

Some feelings/sensations use gustar pattern:

| Expression | Structure | Example |
|------------|-----------|---------|
| I like (enjoy) | Me gusta + infinitive | Me gusta cantar |
| I love | Me encanta + infinitive | Me encanta viajar |
| I hate | Me odia... (rare) | Usually "No me gusta nada" |
| It matters to me | Me importa | Me importa la educación |
| It hurts me | Me duele | Me duele la cabeza |
| It interests me | Me interesa | Me interesa la música |

### Progressive Forms

For ongoing action preferences:

```
English: "I like reading books"
Spanish: "Me gusta leer libros" (preferred)
or: "Me gusta estar leyendo libros" (less common)
```

## Integration with Translation Pipeline

### When Using This Skill

1. **Identify English structure:** "I like [verb/noun]" or "I love [verb/noun]"
2. **Extract the activity/object:** The thing being liked
3. **Determine indirect object:** Based on subject (me, te, le, nos, les)
4. **Determine plural/singular:** Does object require gustan or gusta?
5. **Construct Spanish:** [Pronoun] + [gusta/gustan] + [object]

### Example Translation

**English input:** "I like going to the beach with friends"

1. Structure: "I like [verb]" → Gustar construction needed
2. Pronouns: "I" → "me"
3. Verb: "going" → infinitive "ir"
4. Object: "to the beach with friends" → "a la playa con amigos"
5. **Spanish:** "Me gusta ir a la playa con amigos"

## Quality Checklist

When translating English "like/love" structures:

✓ Used indirect object pronoun (me, te, le, nos, les)
✓ Used "gusta" or "gustan" correctly
✓ Used infinitive form for verbs
✓ Verb agrees with object number
✓ Negation placed correctly (No + pronoun + gusta)
✓ Pronoun placed before verb
✓ Sounds natural to Spanish speaker
✓ Preserves English meaning
