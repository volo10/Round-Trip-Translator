# Binyan Selection Skill (Spanish → Hebrew)

## The Seven Binyanim (Hebrew Verb Patterns)

Hebrew verbs conjugate by 7 different patterns (binyanim), each changing verb meaning.

## Matching Spanish Verbs to Binyanim

| Spanish Verb | Meaning | Binyan | Hebrew | Root |
|--------------|---------|--------|--------|------|
| leer | read (simple) | Pa'al | קרא (kara) | ק-ר-א |
| ser leído | be read | Nif'al | נקרא (nikra) | ק-ר-א |
| enseñar | teach (intensive) | Pi'el | לימד (limad) | ל-מ-ד |
| ser enseñado | be taught | Pual | לומד (lumad) | ל-מ-ד |
| causar | cause | Hif'il | הביא (hevia) | ב-א-א |
| ser causado | be caused | Huf'al | הובא (huba) | ב-א-א |
| luchar | fight (reflexive) | Hitpa'el | התקרב (hitkkarev) | ק-ר-ב |

## Decision Tree

### Simple Action?
```
Spanish: leer, comer, correr, hablar
↓
Binyan: Pa'al (simple active)
Hebrew example: קרא (read), אכל (ate), רץ (ran), דיבר (spoke)
```

### Passive/Happen to Someone?
```
Spanish: ser leído, romperse, perderse
↓
Binyan: Nif'al (passive/reflexive)
Hebrew example: נקרא (was read), נשבר (broke), אבד (was lost)
```

### Intensive/Repeated Action?
```
Spanish: enseñar, estudiar intensamente, servir
↓
Binyan: Pi'el (intensive)
Hebrew example: לימד (taught), למד (studied), שירת (served)
```

### Causing Something to Happen?
```
Spanish: hacer que..., causar, permitir
↓
Binyan: Hif'il (causative)
Hebrew example: הביא (caused to come), הרגיע (calmed), השתתק (made silent)
```

### Reflexive Intense Action?
```
Spanish: argumentar, llamarse, intentar fuertemente
↓
Binyan: Hitpa'el (reflexive intensive)
Hebrew example: התווכח (argued), קרא לעצמו (called oneself), הניסה (tried hard)
```

## Application Examples

### Example 1: Simple Verb
```
Spanish: "Quiero comer pan"
Verb: comer (eat - simple action)
Binyan: Pa'al
Hebrew root: א-כ-ל
Hebrew: "אני רוצה לאכול לחם"
```

### Example 2: Intensive Verb
```
Spanish: "Ella enseña bien"
Verb: enseñar (teach - intensive instruction)
Binyan: Pi'el
Hebrew root: ל-מ-ד
Hebrew: "היא מלמדת טוב"
```

### Example 3: Causative Verb
```
Spanish: "Esto causa problema"
Verb: causar (cause - make something happen)
Binyan: Hif'il
Hebrew root: ג-ר-ם
Hebrew: "זה גורם בעיה"
```

## Common Binyan Patterns

### Pa'al (Simple)
- Most common verbs
- Spanish simple verbs map here
- Meaning: basic action

### Nif'al (Passive/Reflex)
- Passive of Pa'al
- English passive "was done"
- Spanish: "se rompió", "fue roto"

### Pi'el (Intensive)
- More complex/intense action
- Spanish: verbs suggesting effort/repetition
- Example: enseñar (teach) uses Pi'el

### Hif'il (Causative)
- "Make" something happen
- Spanish: "hacer que", "causar"
- Creates transitive from intransitive
