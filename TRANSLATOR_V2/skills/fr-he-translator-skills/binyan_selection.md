# Binyan Selection Skill

## Purpose
Select the correct Hebrew verb pattern (binyan) based on semantic meaning, not French verb form.

## The Seven Binyanim

Hebrew verbs are organized into seven patterns that indicate the type of action:

### 1. Pa'al (פָּעַל) - Simple Active
- Most basic action verbs
- Simple, direct action
- **French verbs mapping:** manger, écrire, lire, aller, voir

| Root | Verb | Meaning |
|------|------|---------|
| כ-ת-ב | כָּתַב | wrote |
| א-כ-ל | אָכַל | ate |
| ה-ל-כ | הָלַך | walked |
| ק-ר-א | קָרָא | read |

### 2. Nif'al (נִפְעַל) - Passive/Reflexive
- Passive voice of Pa'al
- Things happening to the subject
- **French:** être + past participle, se + verb (some)

| Root | Verb | Meaning |
|------|------|---------|
| כ-ת-ב | נִכְתַּב | was written |
| ש-מ-ע | נִשְׁמַע | was heard |
| פ-ת-ח | נִפְתַּח | was opened |

### 3. Pi'el (פִּעֵל) - Intensive/Causative
- Intensive action
- Making something happen
- **French:** verbs with intensive meaning, faire + infinitive

| Root | Verb | Meaning |
|------|------|---------|
| ד-ב-ר | דִּבֵּר | spoke |
| ל-מ-ד | לִמֵּד | taught |
| ב-ק-ש | בִּקֵּשׁ | requested |
| ס-פ-ר | סִפֵּר | told/narrated |

### 4. Pu'al (פֻּעַל) - Passive of Pi'el
- Passive of intensive action
- **French:** être + past participle (intensive)

| Root | Verb | Meaning |
|------|------|---------|
| ל-מ-ד | לֻמַּד | was taught |
| ס-פ-ר | סֻפַּר | was told |

### 5. Hif'il (הִפְעִיל) - Causative
- Causing someone else to do something
- **French:** faire + infinitive, causative constructions

| Root | Verb | Meaning |
|------|------|---------|
| כ-ת-ב | הִכְתִּיב | dictated |
| ש-מ-ע | הִשְׁמִיעַ | made hear |
| ר-א-ה | הֶרְאָה | showed |
| ז-כ-ר | הִזְכִּיר | reminded |

### 6. Huf'al (הֻפְעַל) - Passive of Hif'il
- Passive causative
- **French:** être fait + infinitive

| Root | Verb | Meaning |
|------|------|---------|
| כ-ת-ב | הֻכְתַּב | was dictated |
| ש-מ-ע | הֻשְׁמַע | was made to hear |

### 7. Hitpa'el (הִתְפַּעֵל) - Reflexive
- Action done to oneself
- Reciprocal actions
- **French:** se + verb (reflexive)

| Root | Verb | Meaning |
|------|------|---------|
| ל-ב-ש | הִתְלַבֵּשׁ | got dressed |
| ר-ח-ץ | הִתְרַחֵץ | washed oneself |
| ק-ש-ר | הִתְקַשֵּׁר | called/contacted |
| כ-ת-ב | הִתְכַּתֵּב | corresponded |

## Decision Process

1. **Analyze French verb meaning** (not just the form)
2. **Identify action type:**
   - Simple action? → Pa'al
   - Passive? → Nif'al or Pu'al or Huf'al
   - Intensive? → Pi'el
   - Causative (make someone do)? → Hif'il
   - Reflexive (to oneself)? → Hitpa'el
3. **Select appropriate binyan**
4. **Conjugate for person, gender, number, tense**

## French to Hebrew Mapping Guidelines

| French Pattern | Likely Hebrew Binyan |
|----------------|---------------------|
| Simple verb (manger, lire) | Pa'al |
| être + participe passé | Nif'al/Pu'al |
| se + verb (reflexive) | Hitpa'el (usually) |
| faire + infinitive | Hif'il |
| Intensive action verbs | Pi'el |

## Common Verbs

| French | Hebrew | Binyan |
|--------|--------|--------|
| aimer | אהב | Pa'al |
| parler | דיבר | Pi'el |
| se souvenir | נזכר | Nif'al |
| s'habiller | התלבש | Hitpa'el |
| montrer | הראה | Hif'il |
| être écrit | נכתב | Nif'al |

## Quality Criteria
- Binyan selected based on semantic meaning
- Proper conjugation within chosen binyan
- Person/gender/number agreement maintained
